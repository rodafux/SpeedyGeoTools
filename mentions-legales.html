4. Données Personnelles (RGPD) et Cookies

SpeedyGeoTools respecte votre vie privée :

  • Aucun cookie de pistage : Ce site n'utilise aucun cookie traceur, outil d'analyse d'audience (Google Analytics, etc.) ou pixel publicitaire.
  • Stockage local : Les paramètres de vos logbooks sont sauvegardés de manière transparente via la technologie "localStorage" de votre navigateur web. Ces données restent strictement sur votre appareil et ne sont jamais envoyées ni traitées sur un serveur distant. Vous pouvez effacer ces données à tout moment via le bouton "Effacer l'historique" présent sur la page de création.
  • Génération de PDF : La mise en page pour l'impression est générée intégralement sur votre propre ordinateur ou téléphone.
