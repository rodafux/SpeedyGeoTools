🎒 Antisèche du Géocacheur

📦 Taille des Géocaches

🤏

Nano

< 10ml. Taille d'un écrou, souvent aimantée.

🧪

Micro

< 100ml. Ex : Tube de pellicule, tube PET.

🍱

Small

100ml à 1L. Contient un logbook et petits objets.

🧰

Regular

1L à 20L. Boîte type Tupperware ou munitions.

🪣

Large

> 20L. Grands seaux ou coffres.

🔍 Types de caches

🟢 Traditionnelle : Cache simple située aux coordonnées indiquées.
🟠 Multi-cache : Parcours avec plusieurs étapes (WP) pour trouver le final.
🔵 Mystery / Puzzle : Énigme à résoudre pour trouver les coordonnées.
🌍 EarthCache : Lieu d'intérêt géologique. Pas de boîte, questions à envoyer.
📬 Letterbox Hybrid : Contient un tampon pour votre logbook personnel.
👻 Virtuelle : Lieu à visiter sans boîte physique (photo ou question demandée).
📱 Wherigo : Aventure interactive utilisant une application dédiée.
📅 Event : Rencontre sociale organisée entre géocacheurs.
🧩 Challenge : Mystery demandant d'avoir rempli un défi spécifique.
🏥 Lab Cache : Expérience interactive (Adventure Lab) via smartphone.

📖 Lexique & Jargon

MPLC Merci Pour La Cache (Equivalent de TFTC).
FTF First To Find : Le tout premier à signer le logbook.
STF Second To Find : Le deuxième trouveur.
TTF Third To Find : Le troisième trouveur.
DNF Did Not Find : Log indiquant que vous n'avez pas trouvé.
BYOP Bring Your Own Pen : Prenez votre propre stylo.
GZ Ground Zero : Point exact des coordonnées GPS.
CO Cache Owner : Le propriétaire/créateur de la cache.
TB Trackable : Objet voyageur avec un code de suivi unique.
Molu Moldu : Personne qui ne pratique pas le géocaching.
Spoiler Indice visuel ou textuel révélant l'emplacement.
WP Waypoint : Point de passage ou étape d'un parcours.
Hint Indice supplémentaire pour aider à la recherche.
P&G Park & Grab : Cache rapide d'accès (Drive-in).
CITO Cache In Trash Out : Ramasser les déchets en jouant.
Log Message publié sur le site ou écrit dans le registre.
SWAG Petits objets d'échange trouvés dans les boîtes.
D/T Cotation de la Difficulté et du Terrain (de 1 à 5).
SOS Sign On Site : Action de signer le logbook sur place.
Reviewer Bénévole qui vérifie et publie les caches sur le site.
Aucun terme ne correspond à votre recherche.
